== Description ==

To use this plugin you need to purchase SOHiS software and obtain an id for your hotel.

Easily add an iframe with a page that allows you to create hotel reservations in the SOHiS system.

This plugin relies on our site https://booking.sohis.pl and is used to embed that site in wordpress.
To use plugin (and site) you need to contact with us by e-mail ( sohis@sohis.pl ) or phone ( +48 533 357 221 ).
After contact necessary plugin configuration parameter "place_id" can be obtainted.

Privacy policy (in polish) : https://booking.sohis.pl/other/polityka.pdf

Plugin based on iframe by webvitaly.
